# Skills

A collection of agent skills for Claude Code and other AI coding tools.

## Installation

```bash
npx skills add Phlegonlabs/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [project-long-task](https://skills.sh/Phlegonlabs/skills/project-long-task) | Structured docs workflow for initializing or updating a long-running, milestone-based project |
| [anicenttwo-proj-init](https://skills.sh/Phlegonlabs/skills/anicenttwo-proj-init) | Cross-agent project scaffolding, migration, and AI workflow repair for Claude and Codex |
| [harnass-engineer-start](https://skills.sh/Phlegonlabs/skills/harnass-engineer-start) | Bootstrap harnass-os scaffold into any repo — generates 44 template files, git hooks, and orchestrator state |
| [harnass-engineer-plan](https://skills.sh/Phlegonlabs/skills/harnass-engineer-plan) | Research target repo, align with user, and generate a reviewed milestone + task execution plan |
| [harnass-engineer-final-audit](https://skills.sh/Phlegonlabs/skills/harnass-engineer-final-audit) | Post-production audit across frontend, backend, integration, security, SEO, and release hygiene |
